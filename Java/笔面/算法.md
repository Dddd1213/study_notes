## 二分法

- 左闭右闭
  - left =0
  - right = len-1
  - left<=right
  - right = mid-1下一次查找时不包括mid
- 左闭右开
  - left<right
  - right=mid

- 慢指针：左侧为该数组已经定下来的元素，当快指针不等于target时不更新
