# idea快捷键

- alt + 回车 ：快速创建对象、快速创建测试类
- ctrl + alt + t ：surround with（出现try/catch等快捷键）
- ctrl + alt + o  ：清除所有未使用或无效的导入包
- Ctrl+Alt+h：查看该方法的调用链
- ctrl+Alt+b：查看子类方法实现
- crtl + q ：查看类、方法、属性注释
- ctrl + p ：查看方法的参数
- ctrl + b：查看源码
- ctrl + h：查看接口、实现类快捷键
- ctrl + F12：搜索
- ctrl + shift + r：全局替换
- ctrl + o：生成重写方法
- alt +7：查看类中所有的方法
- alt + enter：快捷生成lambda
